# Initiate Payment — Alfa

When initiating a payment on the Alfa wallet, collect the wallet's `accountNumber` along with the `msisdn` (the customer's mobile number).

---

## Endpoint

| | |
|---|---|
| **Method** | `POST` |
| **Path** | `/v2/wallets/transaction/initiate` |

---

## Request parameters

| Parameter | Required | Type | Length | Description |
|-----------|----------|------|--------|-------------|
| `merchantId` | Yes | Int | 7 | A unique ID provided by Simpaisa to represent each merchant |
| `operatorId` | Yes | Int | 6 | Payment channel ID |
| `amount` | Yes | Int/Float | — | Total amount to be paid by the customer |
| `userKey` | Yes | String | — | A key generated by the merchant to track the transaction |
| `transactionType` | Yes | Int | 2 | Payment type — one-time charge, recurring, or tokenized |
| `msisdn` | Yes | Int | 10 | Customer contact number linked to the wallet account |
| `accountNumber` | Yes | Int | 25 | The customer's Alfa wallet account number — a unique numeric identifier for their wallet (distinct from MSISDN) |
| `productReference` | Yes | String | — | A brief description of the product being sold |

---

## Response parameters

| Parameter | Description |
|-----------|-------------|
| `status` | A Simpaisa response code indicating whether your request succeeded or failed |
| `message` | A human-readable explanation corresponding to the status code |
| `msisdn` | Customer contact number linked to the wallet account |
| `operatorId` | Payment channel ID |
| `merchantId` | A unique ID provided by Simpaisa to represent each merchant |
| `transactionId` | Uniquely generated ID to track every transaction |
